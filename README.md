# Bankist – Minimalist Banking Landing Page

A modern and responsive banking landing page built with **HTML**, **CSS**, and **Vanilla JavaScript**.  
This project focuses on creating smooth user interactions, clean UI design, and modern JavaScript techniques without using any external frameworks.

---

## 🚀 Features

- Smooth scrolling navigation
- Sticky navigation bar
- Modal window functionality
- Tabbed operations component
- Reveal sections on scroll
- Lazy loading images for performance optimization
- Interactive testimonial slider
- Keyboard navigation support
